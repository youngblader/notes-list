## Getting Started

Follow this instruction to run all the system on your machine.

Clone the repo
```
git clone https://github.com/youngblader/notes-list
```

After cloning this repo, open XCode and build project

## Built with
 
* Swift
* UIKit, SnapKit
* CoreData
* MVVM
